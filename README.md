# CatDiet Planner

CatDiet Planner e um app Flutter para gerenciar dieta, rotina diaria e peso de multiplos gatos.

O projeto esta sendo construido com foco em:
- Flutter
- Riverpod
- Hive
- arquitetura `offline-first`

Comandos uteis:

```bash

```

```bash
flutter emulators
flutter emulators --launch <emulator_id>
```
